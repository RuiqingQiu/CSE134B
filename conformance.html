Browser compatibility issues

No issues with table-related tags in Firefox, Chrome, or IE.

No issues with the script tag in Firefox, Chrome, or IE. However, Internet Explorer restricted script.html from rendering Javascript until I clicked a message box to "allow" script.html to show Javascript.

No issues with the script tag in Firefox, Chrome, or IE. In Internet Explorer, I had to restart the browser for turning off Javascript to take effect.

The template tag does not work in Internet Explorer

No issues with the canvas tag in Firefox, Chrome, or IE.

No compatibility issues with forms-related tags in Firefox, Chrome, or IE

No compatibility issues for section tags (body, article, section, nav, aside, h1, h2, h3, h4, h5, and h6, header, footer, address)

Most of the text-level semantics tags are supported by all three browser. However, bdi tag is not supported by all three browsers.
